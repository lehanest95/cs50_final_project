Discord stats explained

I had the hypothesis that the channel in which student first posts a message - is the one which student first experienced real troubles solving it. Therefore I did some Excel + Power BI to generate charts presented at /statistics page.

I used open-source tool DiscordChatExporter to get multiple CSVs from the official CS50 Discord channel. I then wrote Python code "merge_csv.py" to merge CSVs together and form new file "merged.csv" using Pandas. Then this CSV was imported to SQLITE database for ease of queries and speed.

GET data from Discord CS50

You can use DiscordChatExporter to get multiple CSVs from the official CS50 Discord channel.

You can then run "merge_csv.py" file to convert it to merged CSV of all files and create database in which it's gonna be automatically imported.

python merge_csv.py

How to run the Web App in CS50 IDE

Make sure you computer/server/environment has all required modelus listed in "requirements.txt" (IDEs like PyCharm may do it automatically for you). If you run it from CS50 IDE - it has everything installed and works right away.

If you want to run it from your CS50 IDE, do this: Make sure you are in correct directory

cd
git clone https://github.com/lehanest95/cs50_final_project.git
cd cs50_final_project
flask run
